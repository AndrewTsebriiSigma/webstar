## **TIER 1 FILTER QUESTIONS**

Before adding any feature, ask:

| Question | Pass | Fail |
|----------|------|------|
| Does this serve the North Star values? | Enables Identity, Originality, Clarity, Progress, Freedom | Contradicts or dilutes them |
| Does this fit the current version scope? | Within V1-V5 boundaries | Scope creep / wrong version |
| Does this compound on existing features? | Builds on what exists | Isolated / disconnected |
| Is this technically feasible? | Can be built with current resources | Requires fantasy technology |
| Does this solve a real pain point? | Clear problem it addresses | Solution looking for problem |

## **1.0 QUIZ GATE**

*The clarity system — know yourself before you build*

**Solves:** "I build blind, copy others, blend in."

| Quiz | What it discovers | What it gives |
| --- | --- | --- |
| **Quiz 1: Hidden Skills** | Your unique skill combination | Know WHAT makes you different |
| **Quiz 2: Brand Strategy** | Your ideal positioning | Know WHO you serve and HOW |
| **Quiz 3: Visual Language** | Your aesthetic/presentation style | Know HOW to show yourself |

**Output:** Personalized recommendations for building your space

**Methodology:** Iterative design. First 100 users complete quiz + interview. We analyze which questions reveal most differentiation. Quiz evolves based on correlation between quiz answers and profile quality/retention.

**Risk:** Quiz too long → drop-off.
**Mitigation:** Progressive disclosure. Core = 5 min. Deep dives = optional.

## **4.1 CREDITS ECONOMY — HOW IT WORKS**

```
CREDITS ECONOMY MECHANISM

┌─────────────────────────────────────────────────────────────┐
│                                                             │
│   CREATORS                    WEBSTAR                 CLIENTS
│                                                             │
│   Earn Credits ──────────────► Platform ◄────────── Buy Credits
│   (consistent activity)        (middleman)           (real money)
│                                    │                        │
│                              Takes Platform Fee             │
│                                    │                        │
│                                    ▼                        │
│   ◄─────────────────────── Facilitates Exchange ───────────►
│   (receive for services)                        (pay for services)
│                                                             │
│   ✗ NO CASH-OUT                                             │
│   Credits stay in ecosystem                                 │
│   No money transmission issues                              │
│   Value circulates, doesn't exit                            │
│                                                             │
└─────────────────────────────────────────────────────────────┘
```

### **Why This Design**

| Element | Reason |
|---------|--------|
| **Earn through activity** | Consistent effort converts to value |
| **Buy with money** | Clients access creator services |
| **WebSTAR as middleman** | Platform facilitates, takes fee |
| **No cash-out** | Avoids money transmitter licensing complexity |
| **Closed loop** | Value stays in ecosystem, compounds |

### **Credits Risk Note**

| Risk | Reality |
|------|---------|
| **Legal complexity** | CFPB, GENIUS Act, state money transmitter laws apply to virtual currencies. No cash-out simplifies, but legal review required before V4 launch. |
| **Credits feel meaningless** | Credits must unlock real value (services, features, access) or they lose perceived worth. |
